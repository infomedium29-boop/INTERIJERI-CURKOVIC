Upute za GitHub / Cloudflare Pages:
1. Upload sve datoteke iz ovog foldera u GitHub repository.
2. U Cloudflare Pages spoji repository.
3. Build command ostavi prazno.
4. Output directory stavi / ili ostavi prazno, jer je ovo statičan HTML projekt.
5. Deploy.

Napomena:
- Slike su trenutno profesionalni online placeholderi.
- Tekst "VAŠ LOGO" je ostavljen kao privremeni logo.
- Kontakt forma trenutno otvara e-mail aplikaciju s pripremljenom porukom.
- Za produkciju se forma može spojiti na Web3Forms, Formspree ili Cloudflare Function.
